# Rslib project

## Setup

Install the dependencies:

```bash
bun install
```

## Get started

Build the library:

```bash
bun run build
```

Build the library in watch mode:

```bash
bun dev
```
### JSON Configuration

<!-- usage-json-start -->
```json
{
  "mcpServers": {
    "prompt-master": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-prompt-master@0.0.1"
      ]
    }
  }
}
```

<!-- usage-json-end -->

### CLI Usage

<!-- usage-bash-start -->
```bash
npx -y mcp-prompt-master@0.0.1
```

<!-- usage-bash-end -->
